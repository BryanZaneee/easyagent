# Repairbox

**Repairbox** is a Key Bonfire Item in **Dark Souls**. Key Bonfire Items are special items used only at Bonfires or at the Lordvessel within the Firelink Altar. They enable core progression functions, including upgrading equipment, managing inventory, and unlocking fast-travel capabilities to key locations.

> Specialized mortar used to repair weapons and armor by grinding repair powder.
> Rest at a bonfire to make weapon and armor repair possible. Repair powder is fragile, and cannot be taken along

### Usage

* Used to repair broken equipment. Can only be used at a bonfire.
* Only needs to be bought once each playthrough.
* Repairs still cost souls, so there is no difference in price if the box is used instead of a Blacksmith.
* This item does not carry over to New Game Plus.

### Location

* Sold by the Undead Merchant for 3,000 souls.
* Sold by Blacksmith Andre for 2,000 souls.
