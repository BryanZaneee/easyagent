This is a player-created **Build** for Dark Souls. If you want help making your own build, you can consult this useful page on how to make a build by Blaine.

This build includes near-equal Dex / Str to maximize the damage with weapon's that use both stats, this build also has enough Str / Dex to use weapons based ONLY for one of those stats, such as the Great scythe+15, Man-serpent Greatsword +15, Etc.
