# Mage Smith Set

This is an unobtainable armour set in Dark Souls: Remastered. It is worn by Rickert of Vinheim, the blacksmith located in New Londo Ruins near the lift to Firelink Shrine. While the armor is unobtainable, it is in the game's unused assets.
