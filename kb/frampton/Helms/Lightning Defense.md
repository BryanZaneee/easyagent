Lightning Defense is a Stat in Dark Souls.

### Information & Notes

* Defense against lightning attacks.
* See Resistance and Lightning Damage.

Defense against Lightning damage attacks. The following are equipments, items, or spells that increase Lightning Resistance.

### **Armors**

* **Starting Armors**
  + Wanderer Set
* **Light Armors**
  + Great Lord Set
  + Big Hat's Set
  + Chester's Set
  + Dingy Set
* **Medium Armors**
  + Ornstein's Set
  + Silver Knight Set
* **Heavy Armors**
  + Smough's Set

### **Shields**

* Dark Hand (as a shield gives 80% lightning resistance)
* Eagle Shield (75% lightning resistance + other damage types
* Effigy Shield (90% lightning resistance)

### **Rings**

* Thunder Stoneplate Ring
* Speckled Stoneplate Ring
* Blue Tearstone Ring (when health is below 20%)
