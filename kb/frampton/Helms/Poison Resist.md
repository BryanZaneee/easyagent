**Poison Resist** is a Stat in Dark Souls. **Poison Resistance** indicates your defensive power and damage reduction against poison attacks. Defensive Stats are determined by your Attributes and Equipment.

### Information

* Resistance to poison and toxic. Higher value indicates lower tendency to be poisoned.
* Also see Resistance and Poison.

### Poison Defense Equipment

Defense against Poison damage. The following are equipments, items, or spells that increase Poison Resistance.

### **Armors**

* **Armors**
  + **Light Armors**
    - Painting Guardian Set
    - Tattered Cloth Set
    - Gold-Hemmed Black Set
  + **Medium Armors**
    - Catarina Set
    - Channeler's Set
    - Paladin Set
  + **Heavy armor sets**
    - Black Iron Set

### **Shields & Weapons**

* **Shields**
  + Spider Shield (100% block but not for AOE poison)
  + Greatshield of Artorias (100% block but not for AOE poison)
  + Bloodshield
* **Weapons**
  + Gargoyle's Halberd
  + Gargoyle Tail Axe

### **Ring and Others**

* Adding points for any **stats** when leveling up increases the player's **Poison** **Resistance.**
* Poisonbite Ring
