**Bleed Resist** is a Stat in Dark Souls. **Bleed Resist** shows your defense power and damage negation against bleeding attacks. Defensive Stats are determined by your Attributes and Equipment.

### Information

* Resistance to bleeding caused by sharp objects. Higher value indicates lower tendency to bleed.
* Also see Resistance and Bleed.

### Bleed Defense Equipment

Defense against Bleed damage. The following are equipments, items, or spells that increase Bleed Resistance.

### **Armors**

* **Starter Armors**
  + Thief Set
* **Light Armors**
  + Chester's Set
  + Dingy Set
  + Gold-Hemmed Black Set
  + Shadow Set
* **Medium Armors**
  + Catarina Set
  + Channeler's Set
  + Ornstein's Set
  + Paladin Set
* **Heavy armor sets**
  + Smough's Set
  + Black Iron Set

### **Shields**

* **Shields**
  + Bloodshield
* **Weapons**
  + Gargoyle's Halberd
  + Gargoyle Tail Axe

### **Ring and Others**

* Adding points for **Endurance stats** when leveling up increases the player's **Bleed Resistance.**
* Bloodbite Ring
