**Magic Defense** is a Stat in Dark Souls. **Magic Defense** shows your defense power and damage negation against magical attacks. Stats determine the strengths and weaknesses of your character. Defensive Stats are determined by your Attributes and Equipment.

### Information

* Defense against magic attacks.
* Also see Resistance and magic damage

### How to increase

Defense against Magic damage attacks. The following are equipments, items, or spells that increase Magic Resistance.

### **Armors**

* **Starting Armors**
  + Sorcerer Set
  + Big Hat's Set
  + Painting Guardian Set
  + Xanthous Set
  + Dingy Set (Gives the highest magic defense compared to all armor sets when fully upgraded)
* **Light Armor**
  + Channeler Set
  + Paladin Set
* **Heavy Armor**
  + Havel's Set
  + Guardian Set

### **Shields and Weapons**

* **Shields**

  - Crest Shield
  - Havel's Greatshield
  + Crystal Ring Shield
* **Weapons**
  + Moonlight Greatsword (+40 magic defense)
  + Dragon Greatsword (+20 magic defense)
  + Dragon King Greataxe (+20 magic defense)
  + Dragon Tooth (+20 magic defense)
  + Drake Sword (+15 magic defense)
  + Dragon Bone (+15 magic defense)

### **Rings**

* Spell Stoneplate Ring
* Speckled Stoneplate Ring
* Blue Tearstone Ring
