# Armor Smithbox

**Armor Smithbox** is a Key Bonfire Item in **Dark Souls**. Key Bonfire Items are special items used only at Bonfires or at the Lordvessel within the Firelink Altar. They enable core progression functions, including upgrading equipment, managing inventory, and unlocking fast-travel capabilities to key Locations.

> Tool used to etch titanite into armor for reinforcement.
> Rest at a bonfire to make armor reinforcement possible

### Usage

* Allows armor reinforcement at bonfires.
* Only needs to be bought once each playthrough.
* You still need to use souls and titanite as you would if a Blacksmith did the reinforcement.
* This item does not carry over to New Game Plus.

### Location

* Sold by Andre of Astora for 2,000 souls.
* Sold by Blacksmith Vamos for 2,000(?) souls.
