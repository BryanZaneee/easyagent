This is a player-created **Build** for Dark Souls. If you want help making your own build, you can consult this useful page on how to make a build by Blaine.

* Build Name: Royal Tank
* Build Level: 101 (105)
* Build Focus: PvE
* Starting Class: Bandit
* Starting Gift: Master Key
* Build Main Stats: Click
* Link to Dark Souls Calculator: Click here

### Build Equipment

* Weapons:
  + Right Hand
    - Greatsword +15 (lightning +5) / Zweihander Lightning +5 / Demon's Greataxe +15.
    - Choise of the player, any Dragon Weapon, except Priscilla's Dagger.
  + Left Hand
    - Havel's Greatshield
    - nothing.
* Rings:
  + Havel's Ring (you need weight increase)
  + Ring of Steel Protection / Spell Stoneplate Ring / Flame Stoneplate Ring / Thunder Stoneplate Ring (depends of the situation)
  + Covetous Gold Serpent Ring(farming humanity/titanite)
  + Covetous Silver Serpent Ring(soul Farming)
* Armor:
  + First Set to be obtained: Havel's Set
  + Second Set to be obtained: Black Iron Set
  + Third Set to be Obtained: Giant's Set

### Build Strategy

Notes:
-Tank builds are not based on speed but yes on Hard defense. If you like to dodge not the build for you.
-Simple note, tanks need HP so the more HP you got the better. You'll kinda look like Havel in the end until you choose to look like tarkus.
